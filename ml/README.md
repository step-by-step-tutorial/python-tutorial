# Ten Simple Machine Learning Projects

## Prerequisites

* Python
* pip

## Setup

Download and install Python from [official website](https://www.python.org/downloads/).

```shell
# Install pip
python -m pip install --upgrade pip
# Install packages
pip install pandas scikit-learn matplotlib
```
